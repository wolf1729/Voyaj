# 🌍 Voyaj: Your AI-Powered Travel Companion

Voyaj is a premium, high-performance travel platform designed for the modern explorer. Built with **Next.js 15**, **Tailwind CSS**, and **Mongoose**, it seamlessly blends AI-driven destination discovery with an interactive, high-fidelity map experience.

---

## ✨ Features

- **🚀 Exclusive Early Access**: A premium waitlist system with real-time backend integration.
- **🗺️ Interactive Explorer**: High-performance mapping system powered by MapLibre and specialized curvature logic.
- **🎨 Premium Aesthetics**: A "Cyber-Luxury" design system with glassmorphism, animated borders, and ultra-smooth transitions.
- **⚡ Performance-First**: Optimized using Next.js Image components, efficient state management, and minimal client-side hydration.

---

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router)
- **Styling**: Vanilla CSS with Tailwind 4.0 for utility depth
- **UI Components**: Custom-built premium components & Shadcn/UI
- **Feedback System**: [Popcrumb](https://www.npmjs.com/package/popcrumb) (Custom Snackbar Library)
- **Database**: MongoDB with Mongoose
- **Maps**: MapLibre GL JS

## 🏗️ Development

### Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/wolf1729/Voyaj.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```

### Quality Control & Testing

We maintain a strictly guarded codebase with automated local checks:

- **Linting**: Run `npm run lint` to check for code quality and brand alignment.
- **Testing**: Run `npm run test` to execute the Jest suite.
- **Git Hooks**: 
  - **Pre-commit**: Automatically runs `lint` and `test`. Pushes will fail if these checks don't pass.
  - **Pre-push**: Final linting check before code leaves your machine.
